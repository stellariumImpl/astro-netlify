### 注意事项

字符串是用双引号包裹的，会识别转义字符，单引号包裹的是字符，反引号会以字符串的原生形式输出，包括换行和特殊字符；

字符串的字节使用UTF-8编码标识Unicode文本，中文乱码的问题不会再困扰我们了；

字符串一旦被赋值了，就不能被修改了，字符串是不可变的；

字节：1个字节占8bit

字符：
- ascii字符：占1字节
- 常用汉字：占3字节
- 拉丁字母带符号：占2字节
- emoji：占4字节

### 字符串拼接

```go
var str = "hello" + "world"
str += "felix"
fmt.Println(str) // helloworldfelix

var str1 = fmt.Sprintf("%s %s", "hello", "world")
fmt.Println(str1) // hello world

// 多行
var str2 = "hello" + 
  "world" + 
  "felix"
fmt.Println(str2) // helloworldfelix

// 以下做法确实不被允许的
// var str3 = "hello" 
//   + "world"
//   + "felix"

var str4 = `
hello
world
felix
`
fmt.Println(str4) 
// 
// hello
// world
// felix
//
```

- 字符串长度: `len(str)`
- 字符串分割: `strings.Split(str, sep)`
